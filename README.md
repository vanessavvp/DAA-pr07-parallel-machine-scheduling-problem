## **Modo de ejecución**
Ejecutar `make` desde el directorio raíz del proyecto y, seguidamente para ejecutar `make run` el cual ejecutará el [fichero correspondiente a la información de 8 máquinas](./input/I40_8m_S1_1.txt).


De igual manera, en caso de querer ejecutar algún otro fichero de entrada puede hacerlo ejecutando el siguiente comando seguidamente del `make` `./exe input/<nombre-fichero-entrada.txt>`. Por ejemplo:

```console
$ make
$ ./exe input/I40j_2m_S1_1.txt
```
